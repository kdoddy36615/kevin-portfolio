# kevindoddy.dev

Personal developer portfolio showcasing full-stack projects built with modern React, TypeScript, and AI integrations.

**Live:** [kevindoddy.dev](https://kevindoddy.dev)

## Tech Stack

- **Framework:** React 19 + TypeScript
- **Build:** Vite 7
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Testing:** Vitest + React Testing Library
- **Deployment:** Vercel

## Features

- Collapsible sections with smooth CSS grid-rows animation
- Expandable "More about me" bio
- Company and school logos on experience/education cards
- Color-coded skill categories with accent badges
- Responsive layout (mobile → desktop)
- Dark theme

## Featured Projects

### GiftSync
AI-powered gift planning with budget tracking, real-time collaboration, and Supabase auth with Row-Level Security. Built as a Turborepo monorepo with Next.js 15 web app and React Native mobile app.

[Live Demo](https://trygiftsync.app) · [Source](https://github.com/kdoddy36615/giftsync-portfolio)

### MovePlanner
Local-first relocation toolkit with multi-vehicle convoy planning, interactive financial dashboard, and rule-based AI strategy recommendations. Zero-backend architecture using localStorage.

[Live Demo](https://move-planner-demo.vercel.app) · [Source](https://github.com/kdoddy36615/texas-move-planner-portfolio)

### BDO Hub
Personal Black Desert Online progression dashboard with 11 specialized modules including boss tracking, character management, playbooks, and more. Built as a pnpm monorepo with Next.js 16, Supabase backend with 15+ tables, and 12 custom themes.

[Live Demo](https://bdo-hub.vercel.app) · [Source](https://github.com/kdoddy36615/bdo-hub)

## Local Development

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build for production |
| `npm run lint` | Run ESLint |
| `npm test` | Run tests |
| `npm run test:watch` | Run tests in watch mode |
| `npm run preview` | Preview production build |
